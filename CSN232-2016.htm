Welcome to the Spring 2016 CSN-232: Operating Systems (3L-1T-0P) course web page. This course teaches the background of Operating System fundamentals, different functionalities like Process Management, CPU Scheduling Deadlocks, Concurrent Processes, Memory Management, File Systems, and Device Management. The course also includes case study of Windows, UNIX and Linux operating systems. This course requires basic knowledge of computer architecture. (pre-requisite: Computer Architecture and Microprocessors or equivalent).
