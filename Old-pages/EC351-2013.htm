23.07.2013 => Welcome to the Design & Analysis of Algorithms (EC-351/CA-603) class    

 

Welcome to the Autumn 2013 EC 351 course web page. This course teaches techniques for the design and analysis of efficient algorithms, emphasizing methods useful in practice. Topics covered include: sorting; search trees, heaps, and hashing; divide-and-conquer; dynamic programming; graph algorithms; shortest paths; network flow; computational geometry; number-theoretic algorithms; polynomial and matrix calculations; parallel computing; approximation algorithms; and randomized algorithms. This course is an extremely important fundamental course for CS students.
