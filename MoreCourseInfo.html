Tutorial Class Policy

* Attendance is compulsory

* Students can ask questions to clarify doubts about the class lecturer or problem sets (assignments)

* Some example problem types will be solved (if necessary) to demostrate the solving procedures

* For any doubts, students can email the respective T.A.

 

Course Weight Distribution

Problem Sets, Class work / participation in class discussion, Attendance - 5%

Group projects - 20% (10% for continuous evaluation & the rest for final report and presentation)

Mid-terms - 25%

End-term - 50%

Problem Sets / Class Works:

We will distribute 5-6 problem sets (except the problem set on prerequisite) for you to practice your understanding of the course material. The problem sets also provide essential background material for the exams. The problem sets will be graded primarily on an effort basis, but if you do not work through the problem sets you are unlikely to succeed at the exams! We will distribute solutions to the problem sets on the day the problem sets are due to give you instant feedback. Based on the problem sets and class lectures, there can be sudden 1 to 2 in-class quizzes without prior notification.
